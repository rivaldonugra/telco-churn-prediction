# telco-churn-prediction
Project ini membangun model machine learning untuk memprediksi churn pelanggan telekomunikasi berdasarkan karakteristik dan perilaku pelanggan. Analisis dilakukan melalui EDA untuk memahami pola churn, lalu dilanjutkan dengan model klasifikasi. Evaluasi memprioritaskan recall untuk meminimalkan pelanggan churn yang tidak terdeteksi.
